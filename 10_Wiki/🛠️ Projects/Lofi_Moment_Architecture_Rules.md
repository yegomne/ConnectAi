# 🎸 Lo-fi Moment Architecture & Agent Interaction Rules

**Last Updated:** 2026-04-25
**Source:** `e:\Lo-fi Moment\LESSONS_LEARNED.md`

이 문서는 'Lo-fi Moment' 채널 운영 시 다중 에이전트 간의 룰 충돌 및 작업 지연을 방지하기 위한 핵심 아키텍처 원칙입니다.

## 1. 🤖 공식 에이전트 로스터 및 권한 통제
할루시네이션 및 타 에이전트 난입 방지를 위해 다음 5명의 공식 에이전트만 권한을 가집니다:
- **로파이 (Lofi):** 채널 총괄, 메인 프로듀서, 음악(Lyria 3) 전담
- **레이나 (Reina):** 유튜브 알고리즘/SEO 전담 마케터
- **유부장 (Yubuzang):** 파이썬 파이프라인 엔진 및 영상 렌더링 전담
- **이과장 (Lee-Gwajang):** 유튜브 API 연동, 스케줄링 전담
- **박상무 (Park-SangMoo):** 시스템 기획 및 아키텍처 설계
*※ **윤사원**은 오디오 키워드가 나오더라도 절대 등판 불가.*

## 2. 🚀 페이즈 간 완전 자동 바톤 터치 (Baton Touch) 원칙
작업의 단절을 막고 속도를 극대화하기 위해 에이전트 간 릴레이를 강제합니다.
1. **[Phase 2] 로파이 (음악):** 음악 생성이 끝나면 즉시 **레이나**에게 작업을 넘김.
   *※ 사전 조건: 음악 생성(Lyria 3) 전 반드시 3가지 무드를 대표님께 제안 후 선택받아야 함.*
2. **[Phase 3] 레이나 (메타데이터):** 유튜브 SEO 기획이 끝나면 즉시 **유부장**에게 작업을 넘김.
   *※ 하드코딩 매핑: 레이나는 무조건 `youtube-growth-master` 스킬 파일을 참조해야 함.*
3. **[Phase 4] 유부장 (합성/렌더링):** 레이나의 결과를 받아 파이썬 영상 합성 파이프라인 자동 실행.

## 3. 🖼️ 시각물 및 렌더링 절대 원칙
- **커버 이미지 단 1회 생성:** 장르 확정 직후 1번만 생성. 재생성 불가. 
- **경로 고정:** 무조건 `e:\Lo-fi Moment\cover.png` 위치에 덮어쓰기(Overwrite).
- **영상 해상도 보장:** 상하좌우가 잘리지 않게 `1080x1080` (1:1 비율) 렌더링 강제.
- **길이 강제 컷오프:** 쇼츠 영상은 무조건 최대 175초에서 페이드아웃 및 컷오프 (`audio.subclip(0, min(duration, 175.0))`).
- **영문 일기 자막 합성:** 유튜브 메타데이터의 `[Today's Diary]`(100% 영문)를 파싱해 PIL 기반 스크롤링 자막 추가.

## 4. ⚙️ 파이프라인 안정성 및 코어 락 (Core Locks)
- **업로드 전 Human-in-the-loop 강제:** 영상 렌더링 완료 직후 무조건 터미널에서 대표님의 업로드 승인(`y/n`)을 물어 중복 업로드 대참사를 차단.
- **터미널 인코딩 에러 방어:** 파이프라인 입력 및 메타데이터는 무조건 100% 영문으로 작성하여 파워쉘 파이프라이닝 시의 한글 깨짐 방지.
- **동적 파일명 강제:** 렌더링 파일명은 동적 타임스탬프(`_{timestamp}.mp4`) 적용으로 덮어쓰기 금지.
- **Lyria API 락 방어 (Error Case 37):** Lyria API 빈 응답(Empty Response) 시 자동 재시도를 원천 금지하고 즉시 스크립트 강제 종료(`sys.exit(1)`) 후 과금 누수 방지 차단.
