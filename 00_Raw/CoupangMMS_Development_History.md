# 🐻 Coupang MMS/SMS Development History (예곰 단문 메시지 생성기)

## 📌 Project Overview
- **프로젝트명:** 쿠팡 안심번호 단문 메시지(SMS) 자동 분할 & AI 검수 웹 애플리케이션
- **목적:** 쿠팡 안심번호의 SMS 글자 수 제한(약 40~45자 / 80~90바이트)을 우회하기 위해, 긴 문장을 자동으로 SMS 단위로 분할하고 손쉽게 복사하는 유틸리티.
- **아키텍처 (Lean Web App):** 프레임워크 없는 순수 HTML/CSS/Vanilla JS (보안 제약을 피한 로컬 `file://` 실행 환경)
- **주요 기능:**
  - 80 Byte 기준 실시간 바이트 계산 (한글 2byte, 영문 1byte) 및 스마트 문맥 분할
  - 브라우저 로컬 스토리지(Local Storage) 기반 자주 쓰는 상용구(템플릿) 관리
  - 세컨컴(192.168.0.200) 로컬 AI(Ollama - `gemma4:e4b`) 연동을 통한 비즈니스 문구 자동 교정
  - 클릭 한 번에 교정된 텍스트 즉시 적용 및 클립보드 복사

## 🚀 Development Timeline

### 1️⃣ Phase 1: 아키텍처 기획 및 방향성 확립 (박상무)
- **페인포인트 분석:** 안심번호로 80바이트 초과 시 LMS로 전환되며 발송 실패/누락이 발생하는 치명적 문제 접수.
- **구조 설계:**
  - 빠른 유지보수를 위한 '바닐라 JS 기반 정적 웹페이지(1안)'와 '파이썬 RPA 매크로 봇(2안)' 두 가지 아키텍처 제안.
  - 마우스 통제권 상실이라는 파이썬 RPA 봇의 치명적 단점(CS 업무 병목 현상)을 고려하여 **웹 버전(1안)**으로 최종 아키텍처 확정.

### 2️⃣ Phase 2: 예곰 감성 UI/UX 및 핵심 분할 로직 개발 (유부장 & 김대리)
- **UI/UX 디자인:** 예곰 브랜드 특유의 감성(따뜻한 파스텔 오렌지/핑크 톤, 둥근 모서리, 귀여운 폰트)을 녹인 UI 뼈대 완성 (`index.html`, `styles.css`).
- **바이트 계산 알고리즘:** 한글 2byte, 영문/숫자/공백 1byte 기준으로 텍스트를 정확히 계산하는 로직 구현 (`app.js`).
- **스마트 분할 로직:** 단순 글자 수 컷팅이 아닌, 단어가 끊기지 않는 **'스마트 띄어쓰기 분할'** 적용 및 꼬리말에 `[1/3]` 인덱스 자동 부여 기능 구현.

### 3️⃣ Phase 3: 세컨컴 Ollama AI 연동 및 보안/로딩 최적화 (박상무 & 유부장)
- **로컬 AI 도입:** 세컨컴의 남는 하드웨어 자원을 활용하여, 텍스트를 마케터 톤으로 둥글게 교정해주는 `[✨ AI 문구 검수]` 기능 추가.
- **보안 차단(Mixed Content) 우회:** Vercel 배포 시 발생하는 HTTPS ↔ HTTP 통신 차단 문제를 우회하기 위해, 웹 배포를 과감히 취소하고 **로컬(`file://`) 실행 방식**으로 선회.
- **모델 정밀 매칭:** 하드코딩된 `llama3` 모델명을 대표님의 실제 보유 주력 모델인 **`gemma4:e4b`**로 정밀 수정하여 API 404 에러 원천 차단.
- **타임아웃 확장 (60초):** 무거운 모델 사이즈로 인해 세컨컴 VRAM 로딩 시간이 길어지는 현상을 방지하기 위해, Fetch API 타임아웃(AbortController)을 기존 10초에서 **60초(60000ms)**로 대폭 상향하여 안정성 확보.
- **UI/UX 피로도 개선:** AI 검수 결과 클릭 시 뜨던 브라우저 `alert` 팝업을 과감히 제거하여, 불필요한 추가 클릭 없이 즉각적으로 입력창 덮어쓰기 및 클립보드 복사가 완료되도록 '원클릭 자동화' 경험 극대화.

### 4️⃣ Phase 4: 백업 및 실행 인프라 세팅 (이과장)
- **버전 관리:** `git init`, `add`, `commit`을 통한 프로젝트 폴더 로컬 저장소 1차 방어 백업 완료.
- **실행 자동화:** 
  - 다른 엑셀 자동화 툴과 동일한 경험을 위해, 폴더 내 즉각 브라우저 실행용 `실행하기.bat` 파일 생성.
  - PowerShell 시스템 명령어를 통해 윈도우 바탕화면에 **[예곰 단문메시지 생성기] 바로가기(Shortcut)** 아이콘을 꽂아 넣어 클릭 한 번에 구동되도록 인프라 파이프라인 세팅 완료.
- **CORS 및 로컬 파일 보안 이슈 해결 (파이썬 웹 서버 아키텍처 도입):** 
  - 윈도우 `.bat` 파일을 통해 브라우저를 호출할 때 `file://` 환경에서 브라우저가 외부 API 호출을 악성 스크립트로 오인해 차단(Failed to fetch)하는 샌드박스 보안 문제를 근본적으로 해결.
  - 파이썬 내장 모듈(`python -m http.server`)을 활용해 가상의 로컬 웹 서버(`http://127.0.0.1:8500`)를 임시 구동하는 아키텍처를 도입하여 VS Code의 Live Server 환경을 완벽하게 재현.
  - `ping` 명령어를 활용한 백그라운드 지연 실행(Delay Execution) 로직을 적용해, 웹 서버 구동과 브라우저 오픈이 단 한 번의 뱃 파일 더블클릭으로 충돌 없이 완벽하게 자동화되도록 실행 인프라 고도화.

## 🛠 Tech Stack
- **Frontend:** HTML5, CSS3, Vanilla JavaScript
- **AI / API:** Local Ollama API (Model: `gemma4:e4b`) / Fetch API `AbortController`
- **Deployment & Env:** Local File System (`file://`), Git Local Backup, Windows Batch Script (`.bat`, `.lnk`)
