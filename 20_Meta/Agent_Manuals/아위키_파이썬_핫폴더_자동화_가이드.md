# [[아위키 파이썬 핫폴더 자동화 스크립트 가이드]]

## 📌 한 줄 통찰 (Abstractive Summary)
> 파이썬의 Watchdog 라이브러리를 통해 `temp` 폴더를 실시간으로 감시함으로써, 마크다운 파일을 던지는 즉시 A-Wiki 에이전트가 백그라운드에서 자동 기동되는 "완전 자율형 지식 파이프라인"을 완성했습니다.

## 📖 구조화된 지식 (Synthesized Content)

### 실시간 감지 자동화 스크립트 (Python Watchdog)
- **목적:** 사용자가 파일을 `temp/` 폴더에 드래그 앤 드롭하는 즉시 딜레이 없이 Antigravity CLI를 호출하여 A-Wiki 에이전트의 NLP 파이프라인을 가동.
- **주요 로직:** 
  1. `watchdog.observers.Observer`를 사용하여 폴더의 `on_created` 이벤트를 감지.
  2. 감지된 파일이 `.md` 확장자인 경우 `os.system()`을 통해 `antigravity run A-Wiki` 명령어 실행.
- **안전 설정:** 스크립트 파일 자체는 `temp/` 폴더 청소 시 삭제되지 않도록 **`D:\Connect_Ai\awiki_watcher.py`** 경로에 안전하게 저장 및 실행.

## 💎 대체 불가능한 가치 (Unique Value & Expansion)
- 이제 대표님은 번거롭게 터미널을 열거나 챗창에 명령어를 칠 필요 없이, 스마트폰이나 웹 클리퍼를 통해 수집한 텍스트를 `temp/` 폴더로 동기화(Sync)시키기만 하면 됩니다. 24시간 깨어있는 A-Wiki의 감각 기관(Watchdog)이 이를 감지하고 스스로 뇌(Antigravity)를 깨워 대표님의 지식 정원을 가꿉니다. 궁극적인 'Hands-free' 지식화의 완성입니다.

## ⚠️ 모순 및 업데이트 (Contradictions & RL Update)
- **과거 데이터와의 충돌:** 기존에는 수동으로 "작업해"라고 명령해야 파이프라인이 가동되었으나, 파이썬 이벤트 후킹을 도입하여 "자동 사냥(Mode A)"이 완벽하게 구현됨.
- **정책 변화:** `temp/` 폴더 모니터링 방식이 수동에서 실시간 이벤트 기반으로 격상됨.
