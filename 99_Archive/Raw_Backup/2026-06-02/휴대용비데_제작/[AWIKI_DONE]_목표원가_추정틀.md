# 목표 원가 추정 틀 — 국내생산 생수병 결합형 노즐

> **용도:** 물통 없는 국내 브랜드 노즐의 **제조원가 레인지 추정 + 중국 수입품(3,500원)과 비교 + 손익분기·목표 판매가 역산.**
> **성격:** 금형업체 실견적 전 단계의 **계산 템플릿**입니다. 모든 단가는 `[가정]`이며, 실견적이 나오면 그 자리에 넣으면 손익이 자동 계산되는 구조로 짰습니다.
> **주의:** 아래 단가 레인지는 일반적 사출 산업 관행을 바탕으로 한 **추정 범위**일 뿐, 특정 업체 견적이 아닙니다. 반드시 2~3개 업체 견적으로 검증하십시오.

## 2. 제조원가 추정 틀 (가정 단가 — 견적으로 교체)

> 아래 단가는 **추정 레인지**입니다. 물량(MOQ)에 따라 크게 변동합니다. 소량일수록 비싸고, 1만 개 이상부터 단가가 안정됩니다.

```
[ 개당 제조원가(COGS) 추정 ]

① 본체(체결부+대롱) 사출      = [가정] 원   (다규격 정밀 금형, 의료/내열 소재)
② 분사 헤드 ×2                = [가정] 원
③ 보호캡                      = [가정] 원
④ 파우치                      = [가정] 원
⑤ 포장 박스 + 부자재          = [가정] 원
⑥ 조립·검사·국내 인건비       = [가정] 원
───────────────────────────────────
= 개당 제조원가(COGS)          = [합계]

[ 금형비(초기 1회성, CAPEX) ]
다규격 본체 금형              = [견적 필요] 원  (정밀·다단차일수록↑)
헤드 금형(2종)               = [견적 필요] 원
※ 금형비는 개당 원가가 아니라 초기 투자 → 정부지원(디딤돌 R&D 등)으로 충당 검토
※ 금형 감가 분산 = 금형비 ÷ 총생산수량 → 초도 물량이 적으면 개당 부담 큼
```

> **재무 모델러 코멘트:** 핵심 변수는 두 가지 — **MOQ(생산 물량)** 와 **금형비를 정부지원으로 빼느냐**다. 금형비를 자본에서 빼면 개당 원가가 중국 완제품에 근접할 수 있다. 물량이 적으면 개당 원가가 3,500원을 넘을 수도 있음을 직시할 것.

## 3. 중국 수입품과 나란히 비교

| 구분 | 중국 수입품 | 국내 자체 제작(목표) |
| :--- | ---: | :--- |
| 개당 원가 | 3,500원(VAT 포함, 물통 포함) | `[견적 합계]` (물통 없음) |
| 〃 부가세 공제 후(사업자) | 약 3,180원 | `[견적 합계]` |
| 금형비(초기) | 없음(완제품) | `[견적]` → 정부지원 충당 검토 |
| 원가 외 가치 | — | 다규격·안전·국산·특허·환경 |

> **현실 인식:** 국내 노즐 단품이 중국 완제품(3,180원)보다 **싸지긴 어렵다.** 목표는 "더 싸게"가 아니라 **"비슷한 원가에 차별화 가치를 얹어 더 비싸게 판다"** 이다. (경쟁축 = 가격이 아니라 안전·위생·휴대·환경)

## 4. 손익분기·목표 판매가 역산 템플릿

> 제조원가(`COGS`)가 확정되면 아래에 넣어 손익을 계산한다. 부가세는 통과 항목이라 공급가(VAT 제외) 기준.

```
[ 개당 공헌이익 ]
소비자가(VAT 포함)             = [목표가] 원
매출 공급가(÷1.1)             = [계산] 원
(−) 제조원가(COGS)            = [견적] 원
(−) 플랫폼·결제 수수료 [가정 15%] = 공급가 × 0.15
(−) 택배비 [가정] 원          (※ 노즐만이라 물통형보다 부피↓ → 택배비 유리)
(−) 반품·파손 충당 [가정 5%]
(−) 포장 부자재 [가정] 원
──────────────────────────────
= 개당 공헌이익(광고 전)       = [계산] 원

[ 손익분기 판매량 (BEP) ]
고정비(인건비·운영·마케팅 기본) = [작성] 원/월
BEP 수량 = 고정비 ÷ 개당 공헌이익

[ 광고 여력 판정 ]
허용 CAC = 개당 공헌이익 − 목표 개당이익
LTV = (본품 공헌이익) + (헤드·파우치 재구매 공헌이익 × 재구매횟수[가정])
건강 기준: LTV / CAC ≥ 3
```

## 6. 다음 액션 (원가 확정을 위한 실행)

1. **금형·사출 업체 2~3곳 견적** — 본체(다규격)·헤드 금형비 + 물량별 개당 단가. (이게 나와야 위 `[견적]`이 전부 채워짐)
2. **MOQ별 단가 곡선 확보** — 1천/5천/1만/3만 개 시 개당 원가 변화.
3. **금형비 정부지원 매칭** — 디딤돌 R&D·창업패키지로 CAPEX 충당 가능 범위 확인.
4. **택배 계약 단가** — 노즐 부피·무게 기준 실제 택배 단가.
5. **채널 수수료 확정** — 쿠팡/스마트스토어 실제 수수료율.

> 위 1·4·5번 실제 수치를 주시면, 본 템플릿에 넣어 **손익분기 판매량과 필요 광고예산까지 구체 숫자로** 산출해 드리겠습니다.
