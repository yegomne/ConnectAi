# NextEngine Invoice Automation Development History

## 개요
넥스트엔진 주문 샘플 엑셀 파일을 드래그 앤 드롭하여 자동으로 '예곰' 양식 및 '넥스트송장' 양식으로 변환하고, 중복 데이터를 처리하는 윈도우 데스크탑 어플리케이션(PyQt5 기반) 개발 히스토리.

## 🛠️ 개발 연혁 및 주요 구현 기능

### 1. 초기 구조 및 GUI 구현
- **프로젝트 셋업:** `src/` (loader, processor, exporter), `config/` (mapping) 구조로 모듈화 설계.
- **GUI 개발:** `PyQt5`를 활용하여 직관적인 파일 드래그 앤 드롭 인터페이스 구현.
- **데이터 파싱:** `pandas`를 이용한 엑셀 데이터 읽기 및 구조화.
- **오타 교정 로직:** "전자찌"를 "낚시찌"로 자동 변환하는 예외 전처리 기능 추가.

### 2. 파일 변환 및 실행 환경 구성
- **변환 및 자동 저장:** 입력된 넥스트엔진 주문 샘플을 정제 후, 원하는 타겟 폴더에 결과 엑셀 파일들을 저장하도록 로직 구현.
- **실행 편의성 제공:** 코드를 모르는 사용자도 쉽게 프로그램을 테스트하고 사용할 수 있도록 `run.bat` 원클릭 실행 스크립트 제공.
- **백업 처리:** 핵심 기능 구현 중간 단계에서 프로젝트 백업 안정성 확보.

### 3. 데이터 중복 처리 로직 (합포장 처리)
- **수취인 및 주소 동일 (합포장):** 이름과 수령지 주소가 모두 일치하는 주문 건은 최상단의 첫 번째 행(주문)만 남기고 나머지 아래 행들은 자동 삭제(병합) 처리.
- **수취인명 동일 & 주소 다름 (동명이인 처리):** 이름은 같으나 주소가 다른 경우 삭제하지 않고 보존하며, 사용자가 쉽게 식별할 수 있도록 수취인명 셀을 **노란색(`FFFF00`)**으로 하이라이팅 적용.

### 4. 넥스트송장 파일 커스텀 최적화
- **파일 분리 생성:** `예곰[날짜시간].xlsx` 포맷과 더불어 `넥스트송장[날짜시간].xlsx` 포맷 동시 생성 기능 도입.
- **NE 주문번호 단독 추출:** 넥스트송장을 통한 송장 일괄 등록(자동발송처리)을 위해 합포장된 건에 대해서 최상단의 "NE 주문번호"만 깔끔하게 유지하여 출력.
- **헤더(컬럼명) 제거:** 넥스트송장 파일이 시스템에 바로 업로드될 수 있도록 엑셀 출력 시 첫 번째 행의 컬럼명이 나오지 않게 `header=False` 옵션 적용.

### 5. UI/UX 및 데이터 시각화 고도화 (openpyxl 후처리)
- **합포장 데이터 시각화:** 이름과 주소가 동일하여 합포장(중복 제거) 처리된 건들은 최종 출력 파일에서 해당 행 전체를 눈이 편안한 **연한 초록색(`E2EFDA`)**으로 자동 색칠.
- `pandas` 데이터 처리 단계에서 `is_combined` 플래그를 생성하여, 엑셀 저장 직후 `openpyxl` 라이브러리로 서식을 적용하는 파이프라인 완성.

### 6. 포장확인용 파일(팩킹리스트) 시각화 및 상품별 컬러 매핑
- **카테고리 기반 색상 자동화:** 엑셀 상의 원본 상품명이 아닌 `courier_mapping.yaml`에 정의된 '카테고리 키워드'를 기준으로 색상을 할당하여, 동일 카테고리 상품이 원문 이름이 달라도 동일한 색상이 부여되도록 시각적 통일성 확보.
- **생활용품 투명화 (눈 피로도 방지):** 카테고리 매핑이 되지 않은 잉여 상품들은 기본값인 "생활용품"으로 자동 분류되며, 이들은 배경색을 `None(투명)`으로 처리하여 엑셀 가독성 향상 및 눈의 피로도 대폭 완화.
- **합포장 & 다수량 알림 하이라이트:** 합포장된 건이거나 주문 수량이 2개 이상일 경우, "생활용품" 카테고리라 행 배경이 없더라도 해당 중요 알림 셀(수취인명, 수량)에만 핑크/빨강 색상을 부여해 패킹 실수를 원천 차단.

## 📁 주요 디렉토리 및 핵심 모듈
- `gui.py`: 메인 UI, 드래그 앤 드롭 이벤트, 파일 변환 트리거 및 `openpyxl`을 활용한 엑셀 색상 후처리.
- `src/processor.py`: 데이터 필터링, 중복 검사, 오타 수정, 합포장 식별(`is_combined`) 등 핵심 비즈니스 로직.
- `src/pack_processor.py`: 포장확인용 엑셀 변환 로직 및 카테고리 기반 컬러 매핑 처리.
- `src/exporter.py`: `pandas`를 활용한 데이터프레임의 엑셀 출력 및 서식 매핑.
- `run.bat` / `실행하기.bat`: 자동화 프로그램 파이썬 실행 스크립트.
