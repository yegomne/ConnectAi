  * 1:1 깊은 대화에 강점, 다수 외향 영업에 약점
  * 진정성 기반 관계·콘텐츠에 자연스럽게 강함

### 신체 특성
* **체력 약함, 병 없음**
  * 박사장님 1인 운영 + 본업 30분 자동화의 진짜 이유 (체력 보존)
  * 자유 시간 4~6시간이라도 풀가동은 어려운 가능성

### 5가지가 박사장님 14년 사업 패턴을 설명하는 방식

* **14년 카테고리 4번 갈아타며 살아남음** = IQ 134 + 이해도 높음 (새 시스템 적응 능력)
* **전환기마다 빚 누적** = ADD (마무리 약함) + INFJ (영업 약함) → 카테고리 안정화 전 다음 카테고리로 이동
* **50개 빌드 가능** = IQ + 이해도 + AI에게 페이즈 분류해서 지시할 수 있는 시스템 설계 능력
* **외부 사용자 트랙션 0** = ADD (마무리 못함) + INFJ (영업 못함)
* **14년 1인 운영** = INFJ + 체력 (혼자가 가장 효율적인 환경)
* **본인 능력 측정 기준 없음** = 1인 운영 + ADD가 외부 평가받는 마무리를 막아옴 + INFJ가 자기 영업을 막아옴
